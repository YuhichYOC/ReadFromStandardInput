Paiza 攻略用 -- ReadFromStandardInputJ の C++ 版です。
C++ ユーザーは標準入力を使う人多いんかなァ。
まあ、サンプルはどうせ使い物にならんしええやろ ( 適当 )

------- 仕様 -------
サンプルコードは Test01, Test02 および Test03 の 3 種類存在します。
それぞれ
Test01
入力 : 1 行 & 1 アイテム
出力 : string
Test02
入力 : 1 行 & 複数アイテム
出力 : vector<string>
Test03
入力 1 : 複数行 & 複数アイテム
入力 2 : 最初の行の最初のアイテムは入力行数
出力 : vector<vector<string>>
となっています。

現時点で拾った結果はどこにも返さずそのまま標準出力へ表示させています。
各位ご利用の際は string や vector<string> で結果を返させるよう変更を行ってください。

Test03 について
行数の扱いは「行数 & 列数の定義が記載されている最初の行」を含んだ行数です。
コメント中に列数が与えられる前提と書いていますが、結果として読み込み後の vector<vector<string>> は列数を限定せず
入力されたすべての値を保持するようになっています。
